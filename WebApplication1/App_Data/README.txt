League Score Keeper Version 1.0

League Score Keeper is currently in development. This program will allow users to create leagues for various sports that will
contain data about players, matches, teams and other factors for the user to interact with.

This web application will make it simple to use and manage an intramural sports league and declare winners, post stats online
and keep track of team performance.
